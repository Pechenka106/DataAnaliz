# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Cars dataset in Russia" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Анализ ценообразования и прогнозирование стоимости автомобилей на основе их характеристик, а также сегментация автопарка для оптимизации предложения прокатных сервисов и выявления рыночных тенденций.  
**Цель лабораторной:** Аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/card_dataset.csv` — датасет
